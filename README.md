## Trip Expense Manager

Frontend-only React app to track shared trip expenses, see totals, and calculate who owes whom.

### Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the dev server:

   ```bash
   npm run dev
   ```

3. Open the printed local URL (usually `http://localhost:5173`).

### Deploy on Vercel

1. Push this folder as a Git repository to GitHub.
2. In Vercel, create a new project and import the GitHub repo.
3. Framework preset: **Vite** (React).
4. Build command: `npm run build`
5. Output directory: `dist`
